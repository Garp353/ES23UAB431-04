# ES23UAB431-04 - QueMenges

## Descripció del Projecte  
Projecte per muntar una aplicació que permeti als seus usuaris accedir a menús preparats per persones particulars. Fugint d'eines com Glovo o Just Eat que només donen accés a demanar menjar a grans superfícies o cadenes de restaurants.  

En l'aplicació, que s'anomenarà QueMenges, qualsevol persona podrà oferir els seus plats i receptes i els usuaris podran escollir, en temps real, dins d'una gran varietat de plats.

## Estructura del Repositori  
El repositori conté diferents recursos per al desenvolupament del projecte:  

- **`documents/`** → Inclou fitxers rellevants com el PDF d’especificacions.  
- **`requeriments/`** → Conté la fitxa detallada de cada requisit del projecte.  
- **`minutes/`** → Recull les actes de les reunions realitzades.  
- **`others/`** → Carpeta per a arxius auxiliars i altres recursos addicionals.
